Possible Protocol:

List of message:
----------------
Enter into Queue: 			QueueMatch
Request game w/other user: 	RequestMatch
Sync matches with server: 	SyncMatches
Perform move: 				Move
Poll:						Poll
Login:						Login

// Login
Login {
	UserID=<user_id>
	Password=<password>
}

// Add user to queue for game
QueueMatch {
	UserID=<user_id>
	GameID=<game_id>
}

// Request a game with a user
RequestMatch {
	UserID=<user_id>
	GameID=<game_id>
	RequesteeName=<requestee_name>
}

// Poll server for changes for all matches [of a certain game type (optional)]
// as well as new games/requests from other members
Poll {
	UserID=<user_id>
	[GameID=<game_id>]
}

// Send a move to the server
Move {
	UserID=<user_id>
	MatchID=<match_id>
	MoveNumber=<move_number>
	Notation=<move_data>
}

// Returns all move data for given matches to sync local data with server data.
SyncMatches {
	UserID=<user_id>
	MatchIDs=<match_id>[,<match_id>...]
}


Success Messages:

42.1 {
	Successful Login
}

42.2 {
	User added to queue
}

42.3 {
	Game request sent
}

42.4 {
	<poll_data>
}

42.5 {
	Move recorded on server
}

42.6 {
	<match_data>[,<match_data>...]
}

Failure Messages:

666.1 {
	Login failed, invalid username/password
}

666.2 {
	Queue failure
}

666.3 {
	Game request failed
}

666.4 {
	Polling failed
}

666.5 {
	Move failed to record
}

666.6 {
	Failed to sync matches
}
